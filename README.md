# tasadulhemal-dev.github.io

Personal portfolio for **MD Tasadul Islam Hemal** — textile engineer and MSc Innovative Textile Development candidate at Saxion University, working on Digital Product Passports, circular denim, and supply chain traceability.

🌐 Live site: <https://tasadulhemal-dev.github.io>

## What's here

- `index.html` — the entire portfolio in a single, hand-written HTML file (no frameworks, no build step).
- `Resume_Tasadul_Islam_Hemal.pdf` — downloadable CV, linked from the contact section.
- `.nojekyll` — tells GitHub Pages to serve the files as-is without Jekyll processing.

## Stack

- HTML5 + CSS3 (CSS variables, Grid, Flexbox)
- Vanilla JavaScript (Intersection Observer for scroll reveals)
- Google Fonts: Fraunces (variable serif) · Manrope (sans) · JetBrains Mono
- Aesthetic direction: "Raw Indigo Workshop" — deep raw-denim indigo, warm cream paper, burnt copper accents

## Deploying

This is a static site hosted via GitHub Pages from the `main` branch root. To update:

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

GitHub Pages will publish within ~60 seconds at <https://tasadulhemal-dev.github.io>.
